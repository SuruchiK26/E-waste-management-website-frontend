📌 Overview

Electronic waste (E-waste) is one of the fastest-growing global issues. This project aims to create a user-friendly and informative platform where users can:

Understand the importance of proper e-waste disposal

Locate nearby recycling centers

Learn about harmful effects of improper e-waste handling

Submit e-waste for collection or recycling

Raise awareness through blogs and articles

✨ Features

✔️ Home Page with clean UI and awareness content
✔️ About Us Section explaining e-waste problems
✔️ Recycling Center Locator (manual or map-based)
✔️ Contact Form for e-waste pickup
✔️ Educational Blogs/Articles
✔️ Responsive Design for all screen sizes
✔️ Secure Backend (Optional) for form submissions
